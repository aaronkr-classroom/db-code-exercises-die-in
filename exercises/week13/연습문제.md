# 십자말풀이
1. 결정자인 복합키에 포함되는 속성 가운데 단독으로 결정자 역할을 하는 속성이 있는 함수적 종속
> 정답: 부분함수적종속

2. 한 릴레이션에 속한 속성 간의 복잡한 함수적 종속 관계를 쉽게 이해할 수 있도록 그림으로 표현하는 방법
> 정답: 함수종속다이어그램

3. 결정자인 복합키에 포함되는 속성 가운데 단독으로 결정자 역할을 하는 속성이 없는 함수적 종속
> 정답: 완전함수적종속

4. 중복된 데이터 가운데 일부만 갱신되어 데이터의 불일치가 발생하는 현상
> 정답: 갱신이상

5. 릴레이션이 갖고 있는 이상 현상을 발견하기 위해 속성 간의 종속 관계를 규명하는 개념
> 정답: 함수종속성

6. 불필요한 데이터를 함께 삽입하지 않으면 필요한 데이터를 삽입하지 못하는 현상
> 정답: 삽입이상

7. 릴레이션 R에 함수적 종속 A → C가 존재하고, 함수적 종속 A → B ∧ B → C가 성립할 때 존재하는 함수적 종속
> 정답: 이행적함수적종속

8. 어떤 릴레이션 R에 속한 모든 속성의 도메인이 원자 값을 가져야 한다는 제약조건을 만족하는 정규형
> 정답: 제1정규형

9. 관계 DB를 조작할 때 데이터의 중복으로 인해 발생하는 문제
> 정답: 이상현상

10. 어떤 릴레이션 R이 1NF이면서 기본키가 아닌 모든 속성이 기본키에 완전 함수적 종속이어야 한다는 제약조건을 만족하는 정규형
> 정답: 제2정규형

10. 어떤 릴레이션 R이 2NF이면서, 기본키가 아닌 모든 속성이 기본키에 이행적 함수적 종속이 아니라는 제약조건을 만족하는 정규형
> 정답: 제삼정규형

11. 하나의 종속 관계가 하나의 릴레이션으로 표현되도록 릴레이션을 분해하는 과정
> 정답: 정규화

12. 성능에 대한 요구를 만족시키기 위해 데이터 중복과 이상 현상을 대가로 치르면서 보다 낮은 정규형으로 되돌아가는 것
> 정답: 반정규화

> 정답: 2NF \

8. 역정규화에 대한 설명으로 잘못된 것은?

9. 다음 회원관리 릴레이션은 헬스클럽의 회원 정보와 회원 관리를 담당하는 트레이너 정보가 함께 저장된 릴레이션으로, 현재 다음과 같은 데이터가 저장되어 있다. 이 릴레이션에 포함된 이상 현상이 잘못 설명된 것은?
> 정답: 2

10. 9번 문제에 제시된 회원관리 릴레이션의 함수적 종속 관계를 분석하여 다음 표에 정규화하는 과정을 완성하시오.
> 정답:
> * **[FDD]** 회원번호 → (회원이름, 연락처, 가입일자, 담당트레이너, 트레이너이름, 트레이너연락처) / 담당트레이너 → (트레이너이름, 트레이너연락처)
> * **[정규형]** 1NF: O (만족) / 2NF: O (만족) / 3NF: X (불만족)
> * **[릴레이션 분해]** 회원(회원번호, 회원이름, 연락처, 가입일자, 담당트레이너), 트레이너(담당트레이너, 트레이너이름, 트레이너연락처)

11. 멘토링하다 릴레이션의 함수적 종속 관계를 분석하여 다음 표에 정규화하는 과정을 완성하시오.
> 정답:
> * **[기본키]** 멘티_num
> * **[대체키]** 없음
> * **[함수적 종속]** 멘티_num → (멘토_id, 멘토소속, 멘토연락처, 멘토링장소, 멘토링요일, 멘토링시간) / 멘토_id → (멘토소속, 멘토연락처)
> * **[FDD]** 멘티_num → (멘토_id, 멘토소속, 멘토연락처, 멘토링장소, 멘토링요일, 멘토링시간) / 멘토_id → (멘토소속, 멘토연락처)
> * **[정규형]** 1NF: O (만족) / 2NF: O (만족) / 3NF: X (불만족)
> * **[릴레이션 분해]** 멘토링(멘티_num, 멘토_id, 멘토링장소, 멘토링요일, 멘토링시간), 멘토(멘토_id, 멘토소속, 멘토연락처)

12. 동아리가입하다 릴레이션의 함수적 종속 관계를 분석하여 다음 표에 정규화하는 과정을 완성하시오.
> 정답:
> * **[기본키]** 학번, 동아리번호 (복합키)
> * **[대체키]** 없음
> * **[함수적 종속]** (학번, 동아리번호) → (가입일자, 역할) / 동아리번호 → (동아리이름, 동아리지도교수명)
> * **[FDD]** (학번, 동아리번호) → (가입일자, 역할) / 동아리번호 → (동아리이름, 동아리지도교수명)
> * **[정규형]** 1NF: O (만족) / 2NF: X (불만족) / 3NF: X (불만족)
> * **[릴레이션 분해]** 동아리가입(학번, 동아리번호, 가입일자, 역할), 동아리(동아리번호, 동아리이름, 동아리지도교수명)
