# 십자말풀이
1. 특정 칼럼을 분리해서 별도의 테이블로 관리하는 것
> 정답: 수직분할

1. 테이블의 행을 기준으로 분할해서 별도의 테이블로 관리하는 것
> 정답: 수평분할

2. 논리적 데이터 구조(논리적 스키마)를 DBMS가 지원하는 효율적이고 구현 가능한 물리적 데이터 구조(내부 스키마)로 변환하는 과정
> 정답: 물리데이터모델링

3. 자주 사용되는 통계 데이터가 여러 테이블에 산재된 데이터를 기초로 하여 생성되는 경우에 추가할 수 있는 테이블
> 정답: 통계테이블

4. 논리적 설계 단계의 결과물을 기초로 하여 효율적이고 구현 가능한 물리적 DB 구조를 생성하는 과정
> 정답: 물리적설계

5. 트랜잭션 인터페이스 설계를 기초로 하여 구현 단계에서 트랜잭션을 구현하는 데 필요한 세부 사항을 정의하는 과정
> 정답: 트랜잭션설계

6. 포인터를 사용하여 행의 검색을 촉진할 수 있는 DB 객체
> 정답: 인덱스

7. 릴레이션 스키마와 무결성 제약조건을 기초로 하여 정의한 물리적 데이터 구조, 즉 저장 레코드 양식
> 정답: 내부스키마

8. 대용량 테이블의 특정 부분만 주로 사용되는 경우, 해당 부분만 복사해서 추가할 수 있는 테이블
> 정답: 부분테이블

9. 하나 이상의 테이블에 포함된 데이터의 부분집합으로 구성되는 논리적인 테이블
> 정답: 뷰

10. 하나의 업무 단위가 시간의 흐름에 따라 반복적으로 발생하거나 변경되는 이력 또는 진행 이력이 필요한 경우에 추가할 수 있는 테이블
> 정답: 이력테이블
